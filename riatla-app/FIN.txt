🎮 FLUJO DE EJECUCIÓN
════════════════════════════════════════════════════════════════════════════════

1. npm start
   ↓
2. electron electron-main.js
   ├─ Crea BrowserWindow
   └─ Carga file:///riatla-app/index.html
   ↓
3. index.html se renderiza en la ventana
   ├─ Canvas fullscreen
   └─ <script src="renderer.js">
   ↓
4. renderer.js se ejecuta
   ├─ setupScene()       → Three.js + cámara + luces
   ├─ loadVRM()          → Carga riatla.vrm
   ├─ connectWebSocket() → Conecta a ws://localhost:8765
   └─ animate()          → Loop 60 FPS
   ↓
5. Escucha WebSocket por comandos JSON
   ├─ executarComando(comando)
   └─ Actualiza avatar (expresiones/movimientos)
   ↓
6. Panel de debug muestra estado
